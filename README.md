# Spectral Memory — Hermes Agent Plugin

Persistent structured memory for Hermes Agent.

## Install

```bash
hermes plugins install jeannemtl/spectral-memory-plugin9
hermes gateway restart
```

## API key

[spectralmemory.com](https://spectralmemory.com)

hermes config set terminal.env.SPECTRAL_MEMORY_API_KEY $API_KEY
hermes gateway restart
hermes chat

## OR

export SPECTRAL_MEMORY_API_KEY=sm_live_discord_demo
hermes chat
